# Agent Guidelines Index

This folder contains rules and skills for working with CGP user-facing documentation.

## When to Read What

| Situation | Read |
| --- | --- |
| **Starting any task in this repo** | [system-context](rules/system-context.md), [audience](rules/audience.md), [constraints](rules/constraints.md) |
| **Documenting a new feature** | [skills/document-feature](skills/document-feature/SKILL.md), [page-format](rules/page-format.md), [style](rules/style.md), 2–3 sibling pages |
| **Editing an existing page** | [style](rules/style.md), [audience](rules/audience.md), [constraints](rules/constraints.md) |
| **Adding, moving, or deleting pages** | [project-structure](rules/project-structure.md) (`SUMMARY.md` rules) |
| **Writing about a capability you don't know yet** | The corresponding requirement brief in `cgp-documentation-business-requirements/spaces/business/CoverGo-Platform/Requirement-Briefs/` |
| **Unsure about wording, naming, or framing** | [style](rules/style.md), [constraints](rules/constraints.md) |
